# Stream Mini Player (IPTV Mini Player)

A lightweight browser-based IPTV mini player (HLS supported via hls.js).

## Important Notice (Legal / Disclaimer)

This application does **not** provide any TV channels, playlists (M3U), or streaming content.
Users must provide their own playlist URL(s).  
The developer is not responsible for the content accessed through third-party sources.

## Features

- Load M3U playlist via local proxy (CORS)
- LIVE / MOVIES / SERIES tabs
- Group filter + Search
- Favorites (localStorage)
- Remembers last playlist URL (localStorage)
- HLS playback via hls.js

## Local Development

### 1) Install dependencies

```bash
npm install
```
